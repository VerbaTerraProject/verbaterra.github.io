# VerbaTerra Simulation Lab

A minimal, open demo of the VerbaTerra cultural–linguistic engine (vSION series).

**Key ideas**: Cultural inputs (Ritual, Trade, Symbolism, Hierarchy) → Linguistic outcomes (Syntax, Lexicon, Semantics) → **NLIS** (integration) and **CRM** (resilience).

Use the Streamlit app for an interactive exploration, or the Python package for programmatic runs.
